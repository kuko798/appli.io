Appli.io Settings

Gmail sync classifies mail only through the Python service below. The LLM provider is for interview/resume chat features, not inbox labels.

Python email classifier (required for Gmail sync)

Run from repo: cd python_classifier → pip install -r requirements.txt → python service.py (listens on 127.0.0.1:8765). Vite dev dashboard: use http://localhost:5173/appli-classifier if the dev server proxy is enabled.

LLM Provider (interview simulator, resume tools — not Gmail labels)

Local PyTorch server default: http://localhost:8000 (run pytorch_chat_server/server.py; the extension calls /v1 endpoints)
Must match the Hugging Face id loaded by your server (default Qwen/Qwen2.5-1.5B-Instruct). Larger: Qwen/Qwen2.5-7B-Instruct.
Leave blank for local PyTorch server unless you set APPLI_PYTORCH_API_KEY or use a hosted API.
